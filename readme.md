 Чат-бот для мессенджера MAX
 Описание

Чат-бот для мессенджера MAX, автоматизирующий запись клиентов на услугу. Бот реализован на языке Python.

 Функционал

- Выбор услуги из списка через inline-кнопки
- Выбор даты записи
- Выбор временного слота
- Подтверждение или отмена записи
- Навигация назад на каждом шаге

 Структура проекта


max_bot/
├── main.py           # точка входа, запуск вебхука
├── config.py         # настройки из .env
├── handlers.py       # обработчики сообщений и callback
├── keyboards.py      # генерация inline-клавиатур
├── states.py         # состояния конечного автомата
├── data.py           # данные об услугах и расписании
├── .env.example      # пример файла с переменными окружения
├── .gitignore
└── requirements.txt


 Технологии

- Python 3.11
- maxapi — библиотека для API мессенджера MAX
- aiohttp — асинхронный веб-сервер
- python-dotenv — управление переменными окружения
